Projects

Grass Grind

Grass Grind is a 2D pixel-art puzzle game where players guide a lawnmower through custom-designed levels by drawing solid lines on the screen. Using a unique touch-based mechanic, players create paths that the lawnmower follows to clear grass and reach the goal. The game combines strategic planning with interactive control, offering increasingly complex terrain and obstacles. Built with Unity and C#, Grass Grind features mobile-optimized touch input and a nostalgic pixel aesthetic.

Hurf Abad

Hurf Abad is an educational game designed for children aged 5–9, aimed at teaching the Urdu language through three engaging mini-games. With simple controls like arrow keys and colorful, child-friendly visuals, the game creates a fun and accessible learning environment. Built in collaboration with a 2D artist, the project involved integrating custom visual assets to enhance gameplay. Each mini-game focuses on different aspects of Urdu letters and pronunciation, making language learning interactive and enjoyable.

Pharma Game

Pharma Game is a simple 2D click-based game where players repeatedly click on grid boxes in search of a hidden win or lose box. The game features clean visuals and straightforward mechanics, offering quick and casual gameplay. Designed for both PC and Android platforms, it uses a grid system and 2D assets to deliver a smooth and intuitive experience with minimal controls.
